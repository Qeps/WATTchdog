WATTCHdog

{{ motto }}

WATTCHdog is a project that lets you monitor your energy consumption. The system consists of several components: a measurement device based on an STM32 microcontroller with a GSM module for sending notifications and a Wi-Fi module for communication with a local server running on Raspberry Pi 4. The RPi host provides configuration of notifications and the recipients list. Wi-Fi cooperation is set up via a desktop application.
